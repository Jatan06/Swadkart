# Swadkart
---

## ✅ Features

### 👥 User Authentication
- Register as Customer or Admin
- Login/logout
- Session tracking

### 🍽️ Customer Functionalities
- Browse restaurants & dishes
- Add items to cart
- Place and track orders
- Simulate payment
- Give ratings & reviews

### 🛠️ Admin Functionalities
- Add/edit/delete restaurants
- Manage dishes for owned restaurants
- View and update order statuses
- View customer feedback

---

## 🛠️ Tech Stack

- **Java** (Core Java + JDBC)
- **MySQL** (Database)
- **CLI** based menu navigation
- **Design Pattern**: Layered architecture (DAO + Service + UI)
